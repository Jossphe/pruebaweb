Características de Ubuntu Server

Ubuntu Server es una de las primeras distribuciones a implementar cuando presentamos la necesidad de implementar un servidor, ya sea en MYPEs, grandes organizaciones o entornos de desarrollo donde se empleará un uso más rudo.

Presenta múltiples ventajas antes sus alternativas la cual la convierten la distribución estrella entre múltiples administradores de sistemas, desarrolladores y empresas. Ubuntu Server nos presenta un entorno sencillo, robusto y confiable en el cual podemos ejecutar una gran variedad de servicio y aplicaciones configurables a nuestras necesidades.

VENTAJAS

  • Instalación sencilla.

    Ubuntu Server cuenta con un proceso de instalación sencillo y muy bien documentado, el cual facilita la configuración inicial y dudas que puedan surgir durante el proceso.

  • Documentación amplia.

    La comunidad de usuario, desarrolladores que proporcionan soporte y una documentación clara en los distintos foros, páginas oficiales, etc. Facilita el troubleshooting que nos puede presentar durante su uso.

  • Amplio soporte de hardware.

    Es compatible con una amplia variedad de gama de hardware, el cual permite ejecutar una variedad de servidores montados físicamente o en máquinas virtuales.

  • Soporte a largo plazo.

    Recibe constantes actualizaciones que regulan su seguridad y mantenimiento, así mismo ofrece versiones LTS (Long Term Support) el cual nos garantiza un sistema estable, seguro y confiable a través del tiempo.

  • Seguridad, Flexibilidad y Escalabilidad.

    Con herramientas integradas en su instalación inicial, más la facilidad de adaptación según la carga de trabajo requerida, permiten una escalabilidad acorde a su uso, siendo un sistema que nos ofrece confiabilidad y estabilidad.

  • Costo.

    Ubuntu Server es de código abierto y gratuito, lo que lo convierte en una opción costo/beneficio para su implementación, convirtiéndolo en la mejor opción para MYPEs y Startups.

DESVENTAJAS

  • Fuera del estándar empresarial

    Ubuntu es que no es un sistema operativo tan popular como los que son más comerciales debido a que no tiene una fuerte presencia de marca generada por anuncios y campañas de marketing que lo publiciten. Puedes tener problemas de compatibilidad o de adaptación si tu sueles trabajar con este sistema en lugar de uno de los más reconocidos, como Windows.

  • Requiere de conocimientos de programación.

    Aunque no es necesario que poseas grandes conocimientos, si quieres sacarle todo el provecho a Ubuntu, sí que deberías conocer al menos lo básico de los lenguajes de programación, como las etiquetas HTML y propiedades CSS, y, preferentemente, tener conocimientos sólidos en este tema, para así ser capaz de manipular el sistema operativo con mayor control y acceder a las funciones más avanzadas.

  • Limitada debido a las aplicaciones limitadas.

    Una de las limitaciones de Ubuntu es la limitada selección de aplicaciones. Aunque el sistema operativo es gratuito y varias aplicaciones también se pueden descargar de forma gratuita, sus homólogos en Windows y MacOS son considerablemente mejores.

Entornos de escritorios

¿Qué es un entorno de escritorio?

Un entorno de escritorio es un conjunto de software para ofrecer al usuario de una computadora una interacción amigable y cómoda. Es una implementación de interfaz gráfica de usuario que ofrece facilidades de acceso y configuración, como barras de herramientas e integración entre aplicaciones con habilidades como arrastrar y soltar.

Diferentes tipos de entornos de escritorios:

Nombre Concepto
GNOME. Ubuntu utiliza el entorno de escritorio GNOME por defecto. GNOME es una interfaz gráfica de usuario que proporciona paneles en la parte superior e inferior de la pantalla.
Cinnamon Cinnamon es el entorno de escritorio creado por el equipo de desarrollo de Linux Mint esto como un fork de Gnome 2.x, pero con la potencia de Gnome 3.x y como una alternativa para usar en Ubuntu Server.
Mate Mate es un entorno de escritorio derivado del código base de Gnome 2. Nació de la inconformidad de algunos usuarios con la shell de Gnome 3 y que prefirieron quedarse en el modelo que manejaba Gnome 2.
Plasma Desktop Es la primera área de trabajo desarrollada por KDE. Está diseñada para ordenadores de escritorio y portátiles grandes, esta destaca por contar con una amplia gama de configuraciones y que permite desviaciones radicales en el diseño predeterminado.
Xfce Es un entorno de escritorio ligero y su objetivo es ser rápido usando pocos recursos del sistema sin dejar de ser visualmente atractivo y fácil de usar.
LXDE No está diseñado para ser tan complejo como KDE o GNOME, pero es bastante usable y ligero, y mantiene una baja utilización de recursos y energía. A diferencia de otros ambientes de escritorio, los componentes no se integran firmemente. Al contrario, los componentes son independientes, y cada uno de ellos se puede utilizar independientemente con muy pocas dependencias.
