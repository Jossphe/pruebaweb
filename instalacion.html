Instalación de Ubuntu Server.

Requisitos de instalación.

Ubuntu es un sistema operativo inherentemente liviano, capaz de ejecutarse en hardware bastante obsoleto. Canonical (los desarrolladores de Ubuntu) incluso afirma que, en general, una máquina que puede ejecutar Windows XP, Vista, Windows 7 o x86 OS X podrá ejecutar Ubuntu 22.04 incluso más rápido.

Requisitos del sistema para el escritorio Ubuntu 22.04

Ubuntu Desktop tendrá requisitos mínimos más altos que la edición Server de Ubuntu, ya que la versión de escritorio también debe ejecutar una interfaz gráfica de usuario. Los requisitos del sistema a continuación son para el entorno de escritorio GNOME, que es la GUI predeterminada en Ubuntu 22.04.

Esto es lo que Canonical enumera como requisitos mínimos recomendados del sistema para instalar la edición de escritorio de Ubuntu 22.04. Debemos tener en cuenta que esto se refiere a instalaciones físicas. Las instalaciones virtualizadas pueden funcionar con menos.

  1. Procesador de doble núcleo de 2 GHz
  2. 4 GiB de RAM (memoria del sistema)
  3. 25 GB de espacio en el disco duro (o memoria USB, tarjeta de memoria o disco externo, pero consulte LiveCD para conocer un enfoque alternativo).
  4. VGA con capacidad para una resolución de pantalla de 1024×768
  5. Ya sea una unidad de CD/DVD o un puerto USB para el medio de instalación
  6. El acceso a Internet es útil
  7. Recomendado: Tarjeta de video con capacidad de aceleración 3D con al menos 256 MB.

Requisitos del sistema para el escritorio virtualizado Ubuntu 22.04.

Si está instalando Ubuntu 22.04 en un entorno virtual, Canonical dice que su sistema sólo necesita 2 GiB de RAM para funcionar cómodamente. Su página no menciona los requisitos de CPU o almacenamiento, estos requisitos son para instalar Ubuntu 22.04 (edición de escritorio) en una máquina virtual:

  1. Procesador de un solo núcleo de 2 GHz
  2. 2 GiB de RAM (memoria del sistema).
  3. 25 GB de espacio en disco duro.

NOTA: También es recomendable dedicar 128 MB de memoria de vídeo a tu máquina virtual Ubuntu. Si es posible, dedicar un núcleo de CPU adicional, y un GB adicional de RAM aumentará en gran medida la capacidad de respuesta de su máquina virtual, pero no es estrictamente necesario.
VirtualBox: y otros programas de virtualización suelen recomendar 1 GiB de RAM para instalaciones de Ubuntu.
